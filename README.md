# Drag plugin

## Demo: Download and Install

Let's go!

```
$ git clone https://github.com/mailis1000/drag-me.git
$ cd drag-plugin
$ npm install
```

## Development

Run development page on **localhost:8080**

```
$ npm run dev
```

## Build

Build for production.

```
$ npm run build
```

## Run with docker

Run in docker container on port **7766**

```
$ npm run docker-build
$ npm run docker-run
```
